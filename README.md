# 🚀 Bookora – Appointment Booking Management Platform

Bookora is a full-stack appointment booking platform built with the **MERN Stack** that helps service-based businesses manage bookings, customers, and services in a modern digital way.

It is designed for businesses such as salons, clinics, consultants, trainers, tutors, beauty parlours, and other appointment-based services.

# ⚙️ Installation & Setup

## 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/bookora.git
cd bookora
```

---

## 2️⃣ Frontend Setup

```bash
cd Client
npm install
npm run dev
```

---

## 3️⃣ Backend Setup

```bash
cd Server
npm install
npm run dev
```
